# Development

## First time setup

First time setup would require to generate fomantic libs on local

1. Install latest packages

```
yarn install
```

2. Build fomantic libs locally

```
cd ./node_modules/fomantic-ui
```

```
npx gulp install
```

Just press enter on the next instruction

## Run astro locally

```
yarn run dev
```
